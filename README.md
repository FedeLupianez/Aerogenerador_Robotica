# Aerogenerador

## Crear el entorno
crear el entorno virtual de python para instalar platformio
```powershell
python -m venv venv
```

### Activar el entorno :
```powershell
./venv/scripts/activate
```

### Instalar platformio 
```powershell
pip install platformio
```

### Compilar y subir el codigo a la placa : 
```powershell
pio run -t upload
```
